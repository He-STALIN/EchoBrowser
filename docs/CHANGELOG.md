# EchoBrowser Changelog

## [0.1.0] - 2024-01-01

### Добавлено
- ✨ Основная архитектура браузера
- 📑 Система управления вкладками
- 🧭 Панель навигации (адресная строка, кнопки)
- 🏠 Красивая главная страница
- ⚙️ Диалог настроек с тремя вкладками:
  - Основные (поисковая система, главная страница)
  - Приватность (JavaScript, плагины)
  - Внешний вид (тема, размер шрифта)
- 🎨 Две темы (светлая и темная)
- 📋 Система меню (Файл, Правка, Вид, Инструменты, Справка)
- ⌨️ Горячие клавиши (Ctrl+T, Ctrl+W, Alt+Home, F5, F11)
- 📦 Модульная архитектура для простого расширения
- 📚 Полная документация:
  - README.md - руководство пользователя
  - ARCHITECTURE.md - описание архитектуры
  - DEVELOPMENT.md - руководство разработчика

### Технические детали
- **Язык**: Python 3.13+
- **Фреймворк**: PyQt6 6.0.0+
- **Браузер**: PyQtWebEngine 6.0.0+
- **Архитектура**: Модульная с четким разделением ответственности

### Структура проекта
```
EchoBrowser/
├── src/
│   ├── main_window.py      # Главное окно
│   ├── tab_manager.py      # Менеджер вкладок
│   ├── navigation.py       # Панель навигации
│   ├── home_page.py        # Главная страница
│   ├── settings/           # Модуль настроек
│   ├── ui/                 # UI компоненты
│   └── utils/              # Утилиты
├── data/                   # Ресурсы
├── main.py                 # Точка входа
├── dev_run.py             # Dev режим
└── config.py              # Конфигурация
```

## Версионирование

EchoBrowser использует [Semantic Versioning](https://semver.org/):
- **MAJOR** - несовместимые изменения
- **MINOR** - новые функции (совместимые)
- **PATCH** - исправления ошибок

Формат: `v[MAJOR].[MINOR].[PATCH]`

Например: `v1.2.3`
