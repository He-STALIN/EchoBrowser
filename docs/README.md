# EchoBrowser · браузер, который я сделал просто потому что

> «Простой веб-браузер, созданный одним человеком (мной) с использованием ИИ агента. (зачем? не знаю)»

## Что это

EchoBrowser — это минималистичный веб-браузер на Qt (PyQt6), который умеет открывать страницы, показывать вкладки, грузить прогресс, сохранять куки и даже реагировать на полноэкранный режим. Он не пытается заменить Chrome или Firefox, но он мой. И он работает.

## На что способен

- Открывает `HTTP/HTTPS` страницы
- Вкладки (создание, закрытие, переключение)
- Прогресс-бар загрузки в адресной строке
- Полноэкранный режим (да, но пока ломано (остается поле адреса в полноэкранном режиме))
- Куки сохраняются между сессиями (увы, пока через раз, но скоро починим)
- Показывает локальные файлы через `file://`
- Навигация (вперёд/назад, обновление)
- Пользовательский User-Agent (`EchoBrowser/1.0`)
- Тёмная и светлая тема (QSS, но не идеально)

## Что пока не умеет

- Историю посещений (запоминать не хочет)
- Менеджер закладок
- Расширения (их нет и не будет)
- Встроенный поисковик (пока просто Google)
- Свой движок (использует Chromium из Qt)
- Показывать уведомления

## Технические

- Язык: Python 3.13+
- Фреймворк: PyQt6 (`Qt6-WebEngine`)
- Конфигурация: статичные в `config.py`
- Логирование: более точная отладка браузера
- Сборка: `python main.py`
- Зависимости в `requirements.txt`

Подробнее см. в [архитектуре проекта](docs/ARCHITECTURE.md)
