# EchoBrowser - Современный браузер на PyQt6

## Описание

EchoBrowser - это модульный браузер на Python с использованием PyQt6, разработанный с учетом простоты расширения и гибкой архитектуры.

## Особенности

✨ **Базовые функции:**
- 📑 Управление вкладками (открытие, закрытие, навигация)
- 🔗 Адресная строка с поддержкой URL и поиска
- ⬅️➡️ Кнопки навигации (назад, вперед, обновить)
- 🏠 Красивая главная страница
- ⚙️ Полный диалог настроек
- 🎨 Две темы (светлая и темная)
- 📱 Адаптивный интерфейс

## Описание модулей

### 🎯 main.py
Точка входа приложения. Инициализирует приложение и показывает главное окно.

### ⚙️ config.py
Глобальная конфигурация браузера:
- Размер окна
- Пути к ресурсам
- Поисковые системы
- Параметры UI

### 🪟 src/ui/main_window.py
Главное окно браузера. Содержит:
- Менеджер вкладок
- Панель навигации
- Меню
- Обработка горячих клавиш

### 📑 src/ui/tab_manager.py
Управление вкладками:
- Создание новых вкладок
- Закрытие вкладок
- Переключение между вкладками
- Класс `BrowserTab` для отдельной вкладки

### 🧭 src/ui/navigation.py
Панель навигации с:
- Кнопками назад/вперед
- Кнопкой обновления
- Адресной строкой
- Обработкой ввода URL

### 🏠 src/utils/home_page.py
Красивая главная страница браузера с:
- Поисковой строкой
- Информацией о браузере

### ⚙️ src/settings/
Система управления настройками:
- `config.py`: Класс `Settings` для работы с конфигурацией
- `settings_dialog.py`: Диалог настроек с вкладками

### 🛠️ src/utils/
Вспомогательные функции:
- Валидация URL
- Форматирование URL
- Работа с JSON файлами

## Установка и запуск

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Запуск браузера
```bash
python main.py
```

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Ctrl+T` | Новая вкладка |
| `Ctrl+W` | Закрыть вкладку |
| `Ctrl+L` | Фокус на адресную строку |
| `Alt+Home` | Главная страница |
| `F5` | Обновить страницу |
| `F11` | Полноэкранный режим |
| `Alt+←` | Назад |
| `Alt+→` | Вперед |

## Расширение браузера

Архитектура EchoBrowser разработана для простого добавления новых функций:

### Добавление новой вкладки
```python
self.tab_manager.new_tab("https://example.com")
```

### Добавление новой настройки
1. Добавить параметр в `DEFAULT_SETTINGS` в `src/settings/config.py`
2. Добавить UI элемент в `src/settings/settings_dialog.py`
3. Обработать значение в методе `save_and_close`

### Добавление нового меню
```python
# В main_window.py -> create_menu_bar()
new_menu = menubar.addMenu("Новое меню")
action = new_menu.addAction("Действие")
action.triggered.connect(self.your_method)
```

### Создание нового модуля
Добавьте новую папку в `src/` с файлом `__init__.py`:
```python
# src/my_module/__init__.py
from .main_class import MyClass
__all__ = ['MyClass']
```

## Примеры использования

### Загрузить URL в текущую вкладку
```python
current_tab = self.tab_manager.get_current_tab()
current_tab.load_url("https://example.com")
```

### Изменить настройку программно
```python
from src.settings import settings
settings.set("theme", "dark")
```

### Получить значение настройки
```python
search_engine = settings.get("search_engine")
```
